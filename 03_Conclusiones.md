# Conclusiones
A través de este documento hemos realizado la descripción de FUCO, un framework diseñado para la generación de explicaciones contrafacticas en modelos de regresión considerando las restricciones y limitaciones que son impuestas por los dueños y usuarios del modelo. A través de este framework, se consigue generar explicaciones contrafácticas sin tener que adaptar métodos de clasificación. Otra de las ventajas es la definición de umbrales que permite a los usuarios del modelo plasmar sus restricciones y limitaciones. Finalmente, se introduce el concepto de Response sets que permite la identificación de observaciones contrafácticas que pueden ser utilizadas para explicar el modelo.  

Adicionalmente, en el documento se incluye la implementación mediante un caso de uso de FUCO en el entorno de la aeronáutica y aeroespacio. Mediante este caso de uso, se muestra como es la implementación del framework junto a su capacidad de adaptación a diferentes métodos de generación de observaciones. Con el desarrollo de este caso de uso se muestra cómo es posible hacer uso del framework dentro del espectro de la cátedra y cómo es posible aplicar técnicas de explicabilidad a estas áreas del conocimiento.   

Finalmente, como trabajo futuro, se encuentra en el desarrollo del framework mediante la utilización de lógica difusa para poder eliminar las limitaciones que se generan de la utilización de umbrales en el espacio de respuesta. Además, se buscará la implementación de un método de evaluación propio de las observaciones más elaborado al de la ordenación del frente de Pareto mostrada en el caso de uso.
